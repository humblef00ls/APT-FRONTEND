""" # you write code here

Problem Statement

Given three objects, a, b and c, we can only weigh two of them at a time. You want to determine what the weight of all three of them is.

Write function weight3 that is given the weight of the pairs of objects a and b together (parameter ab), a and c together (parameter ac), and b and c together (parameter bc), and returns the total weight of all three objects.

Constraints

  • ab, ac, and bc will be greater than zero.

Examples

  1. ab = 10
ac = 20
bc = 24

returns 27.0
    a must weigh 3, b must weigh 7, and c must weigh 17, so the total weight is 27. 

  2. 
ab = 1.5
ac = 2.5
bc = 3

returns 3.5
    a must weigh .5, b must weigh 1, and c must weigh 2, so the total weight is 3.5. 

  3. 
ab = 200
ac = 300
bc = 200

returns 350.0
    a must weigh 150, b must weigh 50, and c must weigh 150, so the total weight is 350. 

  4. ab = 60
ac = 75
bc = 85

returns 110.0
    a must weigh 25, b must weigh 35, and c must weigh 50, so the total weight is 110
