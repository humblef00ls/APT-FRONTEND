APT: InsideCount

Specification

filename: InsideCount.py def howMany(phrase, letter): ''' Given string parameters phrase and letter, where letter is one character and phrase is a string of words separated by a blank, return the count of the unique words in phrase that have letter in them ''' # you write code here

Problem Statement

Given a phrase of words and a single letter, your task is to determine which of the words in phrase have this letter as one of their letters, and to return the count of the unique words in phrase that have letter in them.

Write function howMany that has two string parameters named phrase and letter. This function returns an integer representing the count of the unique words in phrase that have letter in them. Ignore case for phrase, so the words "That", "that", "ThaT" and "THAT" would all be considered the same word. You can assume letter is always lowercase.

Constraints

  • phrase is of length 1 or more characters.

  • words in phrase are separated by blanks.

  • phrase starts and ends with an alphabetic character, and if there are blanks, there cannot be two adjacent blanks.

  • phrase is only alphabetic characters, upper or lowercase

  • letter is of length 1

  • letter is only an alphabetic character and is lowercase

Examples

  1. phrase = "Duke CompSci graduates are Jedi Masters of the technology sector"
letter = "d"

returns: 3
There are three unique words that contain d: Duke, graduates, and Jedi 


  2. phrase = "This is a very long long long test of this phrase"
letter = "g"

returns: 1
There is only one unique word with "g" in it: long 


  3. phrase = "How are you how are you how are you now"
letter = "h"

returns: 1

There is just one word with h: how



  4. phrase = "How are you how are you how are you now"
letter = "b"

returns: 0

There are no words that have a "b" in them. 


  5. phrase = "Go go go DUKE"
letter = "d"

returns: 1

There is just one word with a d in it. 



  6. phrase = "so"
letter = "s"

returns: 1
