APT: BodyMassIndex

Specification

filename: BodyMassIndex.py def calculateBMI(input): ''' The list parameter input contains two integers in the format [weight, height], where: weight-person's weight (in pounds (lbs)). height-person's height (in inches (in)). The function should return an integer that represents the calculated BMI. ''' # you write code here

Problem Statement

Body mass index (BMI) is a measure of body size, which is often calculated by medical professionals. BMI is calculated based on a person's height and weight. How this calculation depends on the metrics used.

For height in inches and weight in pounds, BMI is calculated as:

(weight/height2) x 703

Given a list of a patient's weight and height, we want to determine a person's BMI. Create the function named calculateBMI that has one parameter named input, which is a list of integers. The list is in the format [weight, height], where weight is the person's weight, and height is a person's height.

This function returns the BMI, based on the input.

Constraints

  • The list only contains one weight and one height.

  • weight is an integer in the range of 1-800.

  • height is an integer in the range of 1-84.

  • The BMI returned is an integer value that rounds down to the closest integer.

Examples

  1. input=[100,60]

Returns: 19
    # A person who is 5ft tall and weighs 100 lbs has a BMI of 19.527777777777775. However, because we're only considering the integer portion, only 19 is returned 

  2. input=[160,65]

Returns: 26
    # A person who is 5ft 5in tall and weighs 160 lbs has a BMI of 26.622485207100592. However, because we're only considering the integer portion, only 26 is returned 

  3. input=[80,50]

Returns: 22
    # A person who is 4ft 2in tall and weighs 80 lbs. 

  4. input=[325,77]

Returns: 38
    # A person who is 6ft 5in tall and weighs 325 lbs.
