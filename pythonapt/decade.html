APT: Decade Nickname

Specification

Filename: Decade.py def nickname(year): ''' year (int) - A year between 1950 and 2019 inclusively Return a string with the nickname of the year in the parameter. ''' return ''

Problem Statement

Write a function nickname that returns a string representing the nickname for the decade in the parameter year. You may assume that year will only be between 1950 and 2019 inclusively. Below is a mapping from each year to its nickname.

Decade Nickname
1950-1959 50s
1960-1969 60s
1970-1979 70s
1980-1989 80s
1990-1999 90s
2000-2009 aughts
2010-2019 twenty-tens

Constraints

  • The int in year is between 1950 and 2019 inclusively.

Examples

  1. year = 2019

Returns "twenty-tens"


  2. year = 2003

Returns "aughts"


  3. year = 1985

Returns "80s"
