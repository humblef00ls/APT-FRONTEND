APT: SentenceLength

Specification

filename: SentenceLength.py def average(sentenceList): """ returns the average sentence length of all the sentences in sentenceList, a list of white-space delimited strings. """ # you write code here return 0.0

Problem Statement

As part of analyzing readability you've been asked to compute the average sentence length of a list of sentences. Each sentence is a a white-space delimited string of words. The length of a sentence is the number of words in it.

See the examples for details.

Constraints

  • sentenceList will have between 1 and 50 strings in it, inclusive of 1 and 50.

  • Each string in sentenceList will contain between 1 and 50 words, words delimited by whitespace.

Examples

  1. sentenceList = ["the", "cat", "ate", "the", "dog"]

returns 1.0
    There are five sentences, each is one word long, average is 1.0. 

  2. sentenceList = ["the cat ate", "the dog"]

returns 2.5
    There are two sentences. One contains three words, the other contains two words. The average is 2.5. 

  3. sentenceList = ["the", "only way", "we can get", "all green is to", "fail"]

returns 2.2
    There are five sentences of length 1, 2, 3, 4, 1. 

  4. sentenceList = ["four score", "and seven years ago", "things were different"]

returns 3.0
    There are three sentences of lengths 2, 4, and 3 (in order that they appear in sentenceList the average is 9/3 or 3.0
