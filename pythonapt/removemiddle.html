APT: Remove Middle

Specification

filename: RemoveMiddle.py def shorten(name): """ return the name with the middle name(s) removed. name has at least one word. """ # you write code here

Problem Statement

Xie needs to be able to shorten names to remove the middle names if there are any.

Write function shorten that given a name returns the name with all the middle names removed.

Constraints

  • name will have at least one word

Examples

  1. name = "Barak Hussein Obama"

returns "Barak Obama"

The middle name is removed. 


  2. name = "Prince"

returns "Prince"

There is no middle name.


  3. name = "Thomas Narten"

returns "Thomas Narten"

There is no middle name.


  4. name = "Elizabeth Rosemond Hilton Wilding Todd Fisher Burton Warner Fortensky Taylor"

returns "Elizabeth Taylor"

All the middle names are removed.
