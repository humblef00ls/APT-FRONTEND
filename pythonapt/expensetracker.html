The function should return a string representing the first month where the total expenses incurred for the year up to that month exceed the yearly limit. ''' # you write code here

Problem Statement

You are always over your yearly budget at the end of the year.In an effort to become more fiscally responsible, you've decided to track your total amount spent, by month. You also want to understand how much you've spent in total at any point in the year.

Given a list of your spending by month and a yearly limit, create the function named expenses that has two parameters named spending (a list of spending, by month)) and limit(the total amount that can be spent for the year).

This function returns a string that represents the first month where the total amount spent for the year (up through that month) exceeds the yearly limit.

Constraints

  • The list named spending should only contain 12 integers.

  • Each item in the list named spending should range between 0 and 700.

  • The minimum amount that your yearly spending can total is 10.

  • The limit must be an integer that is less than the total amount spent over the entire year.

  • The string returned as the month should have the full month name (no abbreviations allowed).

  • The string returned as the month should only have the first letter of the month capitalized (e.g., "January", "February", "March", etc.)

Examples

  1. spending=[10,33,500,0,5,100,210,55,90,100,150,111]
limit=600

Returns: "June"
    # The first month that the yearly limit (600) is exceeded is in June, when 100 is spent for the month, making the total spent through June 648. 

  2. spending=[0,256,321,100,500,0,350,222,90,0,0,0]
limit=1000

Returns: "May"
    # The total spent through May is 1177, which exceeds the 1000 yearly limit. 

  3. spending=[0,0,0,0,200,0,0,0,0,0,0,0]
limit=150

Returns: "May"


  4. spending=[100,100,100,100,100,100,100,100,100,100,100,100]
limit=600

Returns: "July"
