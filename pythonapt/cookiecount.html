APT: CookieCount

Specification

filename: CookieCount.py def countOrders(orders): ''' The list parameter orders contains a string in the format "name:order:quant:name:order:quant:name:order:quant:name:order:quant", where: name-first name of a person. order-the type of cookie ordered. quant-the number of boxes requested. The function should return an integer that represents the maximum number of boxes ordered of any type of cookie. ''' # you write code here

Problem Statement

It's that time of year, where everyone is ordering Girl Scout Cookies. However, the system can only process four orders at a time, in the format

"name:order:quant:name:order:quant:name:order:quant:name:order:quant".

If anyone is interested in buying more than one type of cookie (e.g., Thin Mints and Samoas), then they must submit a separate order for each type of cookie. Thus, each string may contain the same person with multiple orders.

Given a string representing four orders, we want to determine what is the largest quantity ordered in the submission. Create the function named countOrders that has one parameter named orders, which is a string in the format name:order:quant.

This function returns the largest order quantity, based on the input.

Constraints

  • Each name in a name:order:quant sequence is only one name. However, the same name may be included multiple times in the string.

  • Each order in a name:order:quant sequence will contain the name of one type of cookie.

  • Each quantity in a name:order:quant sequence must be at least 1 and no more than 50

  • Each string includes four orders.

Examples

  1. orders="Kim:Thin Mints:2:Erica:Do-si-dos:4:Kim:Thin Mints:6:Karen:Samoas:2"

Returns: 8
    # The largest quantity of cookies sold is Thin Mints, as Kim placed two orders of 2 and 6 boxes. 

  2. orders="Teri:Samoas:2:Erica:Samoas:2:Luke:Samoas:2:Karen:Samoas:2"

Returns: 8
    # Everyone in this batch ordered 2 boxes of Samoas, totaling 8 boxes for Samoas. 

  3. orders="Kim:Tagalongs:4:Erica:Do-si-dos:1:Kim:Tagalongs:3:Karen:Tagalongs:2"

Returns: 9
    # A total of 9 boxes of Tagalongs were ordered. 

  4. orders="Kim:Thin Mints:10:Kim:Lemonades:4:Kim:Lemonades:10:Kim:Samoas:14"

Returns: 14
    # A total of 14 Samoas were ordered.
