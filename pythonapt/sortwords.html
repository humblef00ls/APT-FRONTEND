APT: SortWords

Specification

filename: SortWords.py def sortReverse(phrase): ''' The string parameter phrase is a string of one or more words or combinations of words, numbers, and symbols. The function should return a reverse sorted list of the words in the string. ''' # you write code here

Problem Statement

Given a string, we want to sort the words(in reverse order). Create the function named sortReverse that has one parameter named phrase, which is a string.

This function returns a sorted list of the words from the string in reverse order.

Constraints

  • The string parameter phrase is not empty.

  • Words that are repeated in the string should be counted each time they appear.

Examples

  1. phrase="This is a tester string."

Returns: ['tester', 'string.', 'is', 'a', 'This']
    # "tester" is the largest string in the list. "This" is the smallest string in the list lexicographically. 

  2. phrase="This is A tester is string."

Returns: ['tester', 'string.', 'is', 'is', 'This', 'A']
    # "tester" is the largest string in the list. "A" is the smallest string. "is" appears twice in the list. 

  3. phrase="What time is it?"

Returns: ['time', 'it?', 'is', 'What']


  4. phrase="ARE YOU OLD ENOUGH?"

Returns:['YOU', 'OLD', 'ENOUGH?', 'ARE']
