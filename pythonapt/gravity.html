APT: Gravity

Specification

filename: Gravity.py def falling(time,velo): """ return float indicating number of meters an object has fallen after being dropped/thrown with initial velocity given by float parameter velo (given in meters/second) and after falling for float parameter time seconds """ # you write code here

Problem Statement

Elphaba has decided to try to defy gravity. She's going to drop or throw an object from the top of an infinitely tall building and see how far it falls. She knows exactly what speed she throws the object and has a stop watch she uses to time how long it falls.

Write function falling that returns the number of meters the object has fallen after time seconds have elapsed when the object is thrown with an initial velocity of velo meters/sec.

Ignore any forces due to friction, air-resistance, etc. The infinitely tall building is located in a vacuum. However, the building is on the earth, so accelaration due to earth's gravity should be part of your calculations.

You can find many resources online that give the equation for how far an object falls when dropped. That equation is 0.5*g*t2 where t is time in seconds and g is the gravitational constant which is 9.8 meters/seconds2 (units). When an object is thrown the equation is 

   d = v*t + 0.5*g*t2
Where v is initial velocity in meters/second and t is time in seconds. So simply use that equation, and the value of g = 9.8 in solving this problem.

Constraints

  • time will be greater than zero and less than the number of seconds in a week.

  • velo will be non-negative.

Examples

  1. time = 3
velo = 5

returns 59.1
    If you throw an object at 5 m/sec for 3 seconds it will have fallen 59.1 meters. Note that 5*3 + 0.5*9.8*3*3 = 15 + 44.10 = 59.10 

  2. time = 3
velo = 0

returns 44.1
    If you drop the object instead of throwing it the distance will be 44.1 meters. 

  3. time = 3600
velo = 0

returns 63504000.0
    After an hour a dropped object has traveled 63,504 kilometers or nearly 40,000 miles. 

  4. time = 86400
velo = 0

returns 36578304000.0
    After a day the dropped object has traveled more than 22 million miles
