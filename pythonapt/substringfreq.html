APT: SubstringFreq

Specification

Filename: SubstringFreq.py def frequency(text, n): ''' text (string) - A string with at least one word n (int) - the length of the substring Counts the appearance of each substring in text and returns the most frequent ones as a list. ''' return []

Problem Statement

Considering all substrings of length n within each word separated by whitespace in text, return an alphabetically sorted list of all the most popular substrings.

Constraints

  • n is > 0

  • text is a string with at least one word of length n

Examples

  1. text = 'roses are red'
n = 2

Returns ['re']


  2. text = 'peter piper picked'
n = 1

Returns ['e', 'p']


  3. text = 'the great'
n = 3

Returns ['eat', 'gre', 'rea', 'the']
# All the substrings of length 3 happened equally, notice there is no whitespace in any of these substrings
