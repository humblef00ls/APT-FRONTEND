APT: Badges

Specification

filename: Badges.py def findLabel(labels,deeds,needs): """ return string based on parameters labels a list of strings deeds a list of strings and needs a list of strings """ # you write code here

Problem Statement

You are helping to assign badges based on accomplishments. The accomplishments completed by a participant are specified in the list of strings deeds where each element of deeds is a completed accomplishment.

The names of badges are specified in the list labels. For each element of labels there is a corresponding set of accomplishments in the list needs to earn that corresponding badge, so that earning the badge with name labels[0] (the first element of labels) requires completing each of the accomplishments in needs[0]. Similarly, earning badge with name labels[k] requires completing each of the accomplishments in needs[k].

Return the name of the first badge earned (lowest index), or return the string "nobadge" if no badge is earned. Strings in needs are space-delimited words that indicate the accomplishments needed to earn a badge. See examples for details.

Constraints

  • There will be no more than 20 strings in labels

  • The number of strings of labels will be the same as the number of strings in needs.

  • The string "nobadge" will not appear in labels.

  • Strings in needs consist of space-separated characters.

  • All strings consist only of upper- and lower-case letters 'A'-'Z' and 'a'-'z' or spaces in needs.

Examples

  1. labels = ['wolf', 'bear', 'lion']
deeds = ["fire", "camping", "spanish"]
needs = ["fire knots camping", "spanish fire internet", "fire camping spanish"]

Returns 'lion'
    wolf requires three accomplishments, including knots, which is not in deeds. bear requires internet which is not in deeds. All three accomplishments to earn lion are in deeds. 

  2. labels = ['beginner', 'novice', 'intermediate', 'expert']
deeds = ['skip', 'hop', 'wobble']
needs = ['skip run walk talk', 'wobble teeter fall skip', 
         'skip hop', 'skip hop wobble']

returns 'intermediate'
    Although both intermediate and expert badges can be earned, you must return the first badge (with the lowest index) earned. Since 'talk' is not in deeds and is required for the 'beginner' badge, that badge is not earned. Similarly neither 'fall' nor 'teeter' are in deeds, so 'novice' cannot be earned. 

  3. labels = ['first', 'second', 'third', 'fourth']
deeds = ['code', 'talk', 'plan', 'run']
needs = ['code talk plan think', 'talk plan run think', 'plan run code think', 'run code talk think']

returns "nobadge"
    All the badges require knowing how to think, which is not in deeds, so no badge can be earned. 

  4. labels = ['first', 'second', 'third', 'fourth']
deeds = ['code', 'talk', 'plan', 'run']
needs = ['code talk plan think', 'talk plan run ', 'plan run code think', 'run code talk think']

returns "second"
    Only the second badge doesn't require thinking.
