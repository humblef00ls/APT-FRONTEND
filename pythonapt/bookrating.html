APT: BookRating

Specification

Filename: BookRating.py def calculate(readpages, totalpages, numdays, amazon) """ readpages - number of pages read totalpages - number of pages in the book numdays - number of days read book amazon - book rating on amazon Return the result of the formula shown. Calculate the answer as a float, then return it as a truncated int """ return 0

Problem Statement

Terry has created a new book rating system that is based on 1) the number of pages read (sometimes Terry cannot finish a book) we will call readpages, 2) the total number of pages in the book, we will call totalpages, 3) the number of days Terry read the book, we will call numdays, and 4) the rating of the book on amazon, we will call amazon. Here is the formula Terry has come up with:

Given the number of pages read, the total number of pages in the book, the number of days Terry read the book and the rating of the book on amazon, use Terry's formula to calculate the rating. Terry doesn't like float numbers so after calculating the rating, convert the rating to an integer by truncating it

Constraints/Assumptions

  • readpages > 0

  • totalpages > 0

  • numdays > 0

  • rating > 0.0

Examples

  1.   readpages = 356
  totalpages = 356
  numdays = 10
  amazon = 4.12

  returns 100

  The calculated value is 100.08483601425343. That value is trucated to the
  int 100.


  2.   readpages = 580
  totalpages = 580
  numdays = 50
  amazon = 4.89

  returns 20


  3.   readpages = 250
  totalpages = 267
  numdays = 42
  amazon = 3.1

  returns 22
