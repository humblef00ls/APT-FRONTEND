APT: NotFirst

Specification

filename: NotFirst.py def sortit(letter, words): ''' letter is a string of one lowercase alphabetic letter words is a list of words return a sorted list of the unique words that do not start with letter ''' # you write code here

Problem Statement

Write the function named sortit that has two parameters, letter that is a single lowercase alphabetic letter, and words, a list of words. This function returns a sorted list of the unique words that do not start with letter.

Constraints

  • letter is a lowercase letter, one of a-z

  • words is a list of 0 or more words, each word is a String of one or more lowercase letters from a-z

Examples

  1. letter = "s"
words = ["one", "two", "three"]

returns: ["one", "three", "two"]
 
None of the words start with "s". Note the words returned are sorted. 


  2. letter = "w" 
words = ["one", "two", "three", "wait", "one", "two", "three", "go"]

returns: ["go", "one", "three", "two"]
 
The sorted list contains only unique words that do not start with w. 



  3. letter = "e" 
words = ["duke", "devil"] 

returns: ["devil", "duke"]
 


  4. letter = "d" 
words = ["duke", "devil"] 

returns: []
