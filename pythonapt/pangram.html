APT: Pangram

Specification

filename: Pangram.py def getStats(sentence): ''' sentence (str) - a string with a potential pangram in it For all the letters return a list where value at the kth index is the number of letters that occur k times. ''' return []

Problem Statement

Pangrams are sentences where every letter of the alphabet is used at least once. Some even have each letter appear only once. Implement getStats that takes a sentence and returns a list where the value at the kth index is the number of letters that appeared k times, e.g., the value at index 0 is the number of letters that did not appear at all, index 1 is the number of letters that only appeared once, index 2 is the number of letters that only appeared twice, etc.

Do not count upper and lower case letters as different. And the list should only be as long as needed (it shouldn't end with 0's).

Constraints

  • The sentence may have punctuation, which should be ignored.

  • There are only 26 letters in the alphabet: abcdefghijklmnopqrstuvwxyz

Examples

  1. sentence = 'g dd ee www hhh kkk pppp jjjj'

Returns [18, 1, 2, 3, 2]


  2. sentence = 'Hello and Goodbye'

Returns [16, 6, 3, 1]


  3. sentence = 'Mr. Jock, TV quiz PhD., bags few lynx.'

Returns [0, 26]


  4. sentence = 'The quick brown fox jumps over the lazy dog.'

Returns [0, 20, 4, 1, 1]
