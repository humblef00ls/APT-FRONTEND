APT: LowerVowels

Specification

filename: LowerVowels.py def lower(phrase): ''' This function accepts one parameter, phrase, which is a string. The function should return the number of lowercase vowels in phrase. ''' # you write code here

Problem Statement

We are interested in determining the number of lowercase vowels in a given string.

Create the function named lower that contains one parameter named phrase.

This function returns the number of lowercase vowels present in the string.

Constraints

  • The string parameter phrase should contain at least one character.

  • The number of lowercase vowels present ranges between 0 and 30.

  • Uppercase vowels should not count.

Examples

  1. phrase="Why are you and I still trying to grade?"

Returns: 10
    # There are 10 lowercase vowels in phrase. 'I' is not counted. 

  2. phrase="Angela, Eric, Eve, and I are the same age."

Returns: 12
    # There are 12 lowercase vowels. 

  3. phrase="ARE YOU OLD ENOUGH?"

Returns: 0
    # There are no lowercase vowels present. 

  4. phrase="Pass a pencil."

Returns: 4
