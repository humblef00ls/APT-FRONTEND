APT: ParkingViolations

Specification

filename: ParkingViolations.py def amountOwed(ticket, months): ''' The following parameters are input to the function: ticket-the original ticket amount. months-the number of months that the fine is late. The function should return the current amount owed, which includes the parking fee plus all late fees. ''' # you write code here

Problem Statement

Campus parking has instituted new policies for late parking violations. For every month that a parking ticket remains unpaid, the penalty is 5% of the current balance owed (including prior months late penalties, if applicable).

Create a function named amountOwed that has two parameters named ticket (which is the original ticket amount) and months (which is the number of months that the fine is late).

This function should return the current amount owed for the parking violation, rounding up to the nearest dollar amount (every penny counts!!).

Constraints

  • The original parking fine is between 1 and 200 dollars.

  • The number of months a ticket can be late is 1 to 24 months.

Examples

  1. ticket=100
month=2

Returns: 111
    # The amount owed in fines is $110.25. However, because campus parking rounds up, the total amount owed is $111. 

  2. ticket=25
month=15

Returns: 52


  3. ticket=50
month=5

Returns: 64


  4. ticket=75
month=6

Returns: 101
