APT: Max Order

Specification

filename: MaxOrder.py def forItem(orders, item): ''' orders (list of strs) - list of orders for items in format "ITEM:QUANTITY" item (str) - item of interest in ORDERS Return in a list the largest ordered quantity for ITEM in ORDERS and how many times that quantity was ordered. ''' return [0, 0]

Problem Statement

Write the function forItem that takes orders, a list of strings that represent orders, and item, a string for a specific item. Each element in orders is a string of the format 'item:quantity' where quantity is an integer. This function then returns the largest quantity ever ordered for the parameter item and how many times the maximum order quantity of item occurs in orders.

If item is not in orders, return a max quantity of 0 and a count of 0.

Constraints

  • The quantity for an order will be a positive integer.

Examples

  1. orders = ['apple:1']
item = 'apple'

Returns [1, 1]


  2. orders = ['banana:4', 'orange:22', 'banana:4', 'banana:3']
item = 'banana'

Returns [4, 2]


  3. orders = ['apple:1', 'banana:2']
item = 'orange'

Returns [0, 0]
# 'orange' is not in orders, therefore it returns a max of 0 and a count of 0
