APT: CharityDonor

Specification

Problem Statement

You are given information about donors and their contributions. Return the name of the most generous donor, the donor who has given the most to charity.

A donor may give more than once. Determine the name of the donor who has given the largest total amount to charity. If there is more than one donor with the largest amount, return the name that comes first in alphabetical order.

For example, if the donor list is the following you'll return "Blue".

  1. "Sun:70.00"
  2. "Zebra:80.00"
  3. "Blue:30.00"
  4. "Edwards:50.00"
  5. "Blue:50.00"

Both "Zebra" and "Blue" gave the most amount, $80.00. Return "Blue" as it comes before "Zebra" alphabetically.

A donation is represented as a string with the name of the donor first followed by ":" as the separator, followed by the dollar amount given. The dollar amount will represent a valid float.

Given a list of donations (a list of strings), return the name of the donor with the largest donation. If there is more than one donor with the maximal amount, return the name of the donor that comes first alphabetically.

Constraints

  • contributions is a nonempty list of strings.

  • Each character of the strings in contributions will be a lowercase letter 'a'-'z', upper case 'A'-'Z', digits '0'-'9', '.' or a ':'.

  • The format of each string in contributions is "name:amount" where amount is a valid flaot. That is, there will be exactly one ':' in each string of contributions. To the right of the ":", the remaining characters in the string will be in the format of a valid float.

Examples

  1. ["Zack:800.00", "Gregory:900.00"] 

Returns "Gregory"


  2. ["Zack:800.00", "Gregory:800.00"] 

Returns: "Gregory" 


  3. ["Sun:70.00", "Zebra:80.00", "Blue:80.00", "Edwards:50.00"] 


Returns: "Blue" 


  4. ["Sun:70.00", "Blue:60.00", "Zebra:80.00", "Edwards:50.00"] 


Returns "Zebra" 


  5. ["Zack:800.00"] 


Returns: "Zack" 


  6. ["French:30.00","Zack:10.00", "Tie:20.00", "Zack:10.00", "Mack:20.00", 
"Zack:10.00", "Green:10.00", "Zack:10.00", "Mack:10.00"]

Returns "Zack"
