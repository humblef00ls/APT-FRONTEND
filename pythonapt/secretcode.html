APT: SecretCode

Specification

Filename: SecretCode.py def decrypt(message, numbers): ''' message (string) - regular message with 1 or more words numbers (list of ints) - list of positive and negative integers Return the secret message hidden in message using the list numbers. ''' return ''

Problem Statement

Write the function decrypt that finds a secret message. It does this by building a string from the words in the string message and a list of positive and negative integers in the list numbers. Build the secret message by indexing into message's kth word using the kth number in numbers.

If the number cannot index into the word, then do not use that word as part of the secret message. A number is either positive or negative and cannot index into the word if it is >= len(word) or < -1 * len(word).

Constraints

  • There is at least one word in message.

  • The variable numbers is a list of positive and negative integers and will be the same length as the number of words in message.

Examples

  1. message = "the great and terrible monster"
numbers = [1,2,0,2,4]

Returns 'heart'


  2. message = 'roses are red violets are blue'
numbers = [0,0,0,-9,3,-1]

Returns 'rare'


  3. message = 'Hello, it looks as if the bear symbol is all over campus. Why do they like the bear symbol, George?'
numbers = [0, 0, 10, -4, 4, 0, 3, 4, 5, -4, -10, 4, 5, -6, -7, 6, 4, 0, 5, 1]

Returns 'Hitrouble'
# This is "Hi trouble" Notice the secret message will have no spaces.
