APT: Midpoint

Specification

filename: Midpoint.py def getPoint(coordinates): ''' The function accepts one parameter named coordinates, which is a list containing two strings in the format ["x1,y1", "x2,y2"], where: "x1,y1" is an (x,y) coordinate pair for the first point "x2,y2" is an (x,y) coordinate pair for the second point The function should return a list containing the x and y coordinates of the midpoint in the format [x,y]. ''' # you write code here

Problem Statement

The midpoint between any two points on the Cartesian plane can be calculated using the formula:

((x1+x2)/2, (y1+y2)/2) where (x1, y1) are the coordinates for the first point and (x2, y2) are the coordinates for the second point.

Create a function named getPoint that has one parameter named coordinates (which is a list of two strings in the format ["x1,y1", "x2,y2"]).

The function returns a list that contains two doubles [x,y], which correspond to the x and y coordinates of the midpoint.

Constraints

  • All coordinates for the two points used to calculate the midpoint are integers between -30 and +30.

  • The coordinates of the midpoint are expressed as double values.

  • There is no space before or after the comma in each coordinate.

Examples

  1. coordinates=["2,2", "2,2"]

Returns: [2.0, 2.0]
    # The midpoint between two points that are the same is the same as the two points. Note this includes a double representation of the midpoint. 

  2. coordinates=["-2,2", "2,2"]

Returns: [0.0, 2.0]
    The midpoint occurs at (0.0, 2.0). 

  3. coordinates=["-5,-10", "4,3"]

Returns: [-0.5, -3.5]


  4. coordinates=["1,0", "-10,-5"]

Returns: [-4.5, -2.5]
