APT: Consensus

Specification

Filename: Consensus.py def majorityWinner(voteA, voteB, voteC): ''' voteA (int) - Number of votes received by candidate A voteB (int) - Number of votes received by candidate B voteC (int) - Number of votes received by candidate C Returns a string representing which candidate won the majority vote, based on the three parameters. If there is no majority winner, return "No one". ''' return ''

Problem Statement

Winning the majority vote means receiving strictly more than half of the votes. Write a function that determines who won the majority vote among three candidates A, B, or C. If no one won the majority vote, return the string "No one" (note: A, B, C, and N are capitalized).

Constraints

  • Every candidate gets at least one vote.

Examples

  1. voteA = 6
voteB = 3
voteC = 1

Returns "A"


  2. voteA = 5
voteB = 4
voteC = 1

Returns "No one"
# There are a total of 10 votes so a candidate must win 6 or more votes to get a majority


  3. voteA = 1
voteB = 1
voteC = 1

Returns "No one"
