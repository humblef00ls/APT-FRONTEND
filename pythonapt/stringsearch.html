APT: StringSearch

Specification

filename: StringSearch.py def findString(input): ''' The string parameter input is comprised of one or more words or combinations of words, numbers, and symbols. The function should return the number of words in the string that end with a vowel. ''' # you write code here

Problem Statement

Given a string, we want to identify all the words contained in the string that end with a vowel. Create the function named findString that has one parameter named input, which is a string.

This function returns the number of words in the string that contain end with a vowel.

Constraints

  • The string parameter input cannot be empty.

  • Words in the original string input that end with a vowel, but also are followed by a symbol (e.g., '.' or ',' or ':' or ';' should not be included in the final count.

  • Words that are repeated in the string should be counted each time they appear.

Examples

  1. input="Hey you."

Returns: 0
    # The last word is 'you.'. However, because a '.' follows the 'u', then it is not the last character in the substring. Thus, it is not included, and the count is 0. 

  2. input="Hey you"

Returns: 1
    # The last word is 'you' and it is the only substring that ends in a vowel. 

  3. input="Are you there? I can't hear you"

Returns: 4
    # 'there?' is not included because '?' is the last character in the substring. Also, 'you' appears twice in the sentence, and is thus counted twice. 

  4. input="cat $13ha FIVE yellow submarine"

Returns: 3
    # '$13ha' is included in the count because the substring ends in a vowel.
